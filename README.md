# Cisco Network Configuration Project

## Description
Ce projet présente la configuration d’un réseau dans Cisco Packet Tracer avec plusieurs sous-réseaux reliés à un routeur central.  
L’objectif est de démontrer la configuration d’adresses IP, l’interconnexion de plusieurs segments réseau et les tests de connectivité entre les équipements.

## Topologie du réseau
Le réseau contient :

- 1 routeur Cisco 2911
- 3 commutateurs Cisco 2960-24TT
- 3 serveurs
- 3 postes clients (PC)

## Sous-réseaux utilisés

### Réseau 1
- Adresse réseau : `192.168.50.0/26`
- Masque : `255.255.255.192`

### Réseau 2
- Adresse réseau : `192.168.50.0/27`
- Masque : `255.255.255.224`

### Réseau 3
- Adresse réseau : `192.168.50.64/27`
- Masque : `255.255.255.224`

## Travaux réalisés
- Création de la topologie réseau dans Cisco Packet Tracer
- Connexion des PC, serveurs, commutateurs et routeur
- Attribution des adresses IP aux différents équipements
- Configuration des interfaces du routeur
- Mise en place de la communication entre plusieurs sous-réseaux
- Vérification de la connectivité réseau avec des tests de type `ping`
- Validation du bon fonctionnement de l’infrastructure

## Compétences démontrées
- Configuration de base d’un routeur Cisco
- Configuration IP sur des équipements réseau
- Compréhension du sous-réseautage
- Utilisation de Cisco Packet Tracer
- Dépannage de connectivité réseau
- Organisation d’une topologie réseau simple

## Outils utilisés
- Cisco Packet Tracer

## Objectif pédagogique
Ce projet m’a permis de pratiquer la configuration d’un réseau multi-segments, de mieux comprendre le routage entre sous-réseaux et d’appliquer les bases de l’administration réseau avec des équipements Cisco.
